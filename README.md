# 게임엔진기초_팀10 - UI(전지우)

## 구현 기능
* 메인 메뉴(타이틀)
   * 저장 및 불러오기
   * 커서 조작
   * 게임 종료
* 설정 화면
   * 메인 메뉴와 연결
   * esc를 눌러 뒤로가기
   * 커서 조작
* 대화 화면
   * 게임 시작 시 화면 진입
   * S_Line 형식의 애셋을 한 줄씩 불러와 이름, 칭호, 대사 등으로 분해해 출력
   * 스페이스바나 화면 하단의 버튼을 눌러 대사 넘기기
   * 커서 조작
* 육성 화면
   * 화면 진입
   * 커서 조작

---

## 파일 설명 (/Contents/UI)
* **UI 애셋**을 저장함.
* **WBP_UI**: 화면에 띄워지는 UI의 모양과 그 동작을 정의함.
* **S_Line**: 대화 단계에서 사용되는 대화를 저장하는 데이터 형식을 정의함.
* **Line_Tutorial**: 디버그용으로, 튜토리얼의 대사를 저장하는 S_Line 형식의 데이터.
 
## 파일 설명 (Source/GameEnTeam10_Project)
* **CustomSaveGame**: SaveGame 클래스를 상속하는 클래스로, 게임 세이브 파일에 저장될 변수를 정의함.
